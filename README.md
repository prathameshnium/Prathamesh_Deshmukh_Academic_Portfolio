This repository hosts the source code for my **Academic Portfolio**. It serves as a central hub for my research outputs, including software publications, experimental datasets, and hardware control protocols.

- **Research Focus:** Synthesis of multiferroic composites, low-temperature characterization, and development of open-source instrumentation tools.

### Scientific Software & Projects

A selection of developed tools for laboratory automation and physics simulations.

| Project | Status / Description | Stack |
| :--- | :--- | :--- |
| [**PICA Automation Suite**](https://github.com/prathameshnium/PICA-Python-Instrument-Control-and-Automation) | **Open Source (v1.0.1)** <br> *Submitted to Journal of Open Source Software (2025).* <br> A modular PyVISA framework for high-precision transport measurements (IV, RT, C-f). | `Python` `PyVISA` `Multiprocessing` |
